Simple Header

Target Audience

The target audience for this header will be young to middle-age professionals looking for a career or more information about the company

Client & User Needs

The 'Client' and 'User' needs are similar. They both want the site navigation to be clear and the main action of viewing available jobs given high visual importance.

Design Critique

This header is a very simple layout that use color to show important and active items. The color orange is a bright attention catching color but it also can be used in a professional stance because the color represents success and creativity.

Contact Information and Form

Target Audience

This contact form can work for any person trying to reach a company. It caters to older generations by providing physical contact information and to younger generations by giving an online means.

Client & User Needs

The needs for both groups it to have the user contact the company by the proper means set by the Client.

Design Critique

The form uses proximity and spacing to show similar items in a group. For example the telephone numbers are close together while the space between the email address and phone numbers is large. It uses color and image to separate physical and online contact means. The images of the map and icons like the phone enhance understanding of the written text. The use of color blue gives a feel that you can trust contacting this company.

Pricing Table

Target Audience

I believe this item could be used for all persons trying to find information on a service quickly.

Client & User Needs

The 'Client' is wanting the user to sign-up for the various packages but has one seen as a better deal. The 'User' needs to have a quick summary of the package details and overall price. They then want easy access to sign-up.

Design Critique

The design uses one vibrant color to highlight the item the Client wants the user to focus on. It also uses the Gestalt Principle of proximity to define the items in each pricing package. It also has smaller spacing between the pricing tables than it does with the rest of the page indicating that they all are items to be compared with each other.
